# Assignment-5
This project contains two Python programs demonstrating basic data structure concepts such as dictionaries and list slicing.
Task 1: Create a Dictionary of Student Marks
This program creates a dictionary where student names are stored as keys and their marks are stored as values.
The user is asked to enter a student's name. If the name exists in the dictionary, the program displays the student's marks. Otherwise, it shows a message that the student was not found.
Concepts Used
Python Dictionary
User Input
Conditional Statements
Task 2: Demonstrate List Slicing
This program demonstrates the concept of list slicing in Python.
A list containing numbers from 1 to 10 is created. The program extracts the first five elements from the list and then reverses them. Finally, both the extracted and reversed lists are displayed.
Concepts Used
Python Lists
List Slicing
List Reversal
